Hello!
This website is a group effort to document films and lists of films in one easy to access place. We are pulling information from the APIs of IMDB, TMDB, and Youtube.

These APIs are being used to add movie information to a Dynamo Database hosted on AWS servers. The user is able to search up a film to obtain general information about the film, along with a trailer and poster. Once a film is searched with our UI, you can then search up the film by its given IMDB ID, which will allow you to pull the information directly from the Dynamo Database where it is being stored.

Feel free to explore and contact us via the 'Contact' page if you have any questions or comments!
